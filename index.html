¿CÓMO FUNCIONA?

La Public Key se configura acá y va a Firestore. El Access Token va directo a Vercel como variable de entorno — nunca pasa por el browser. Cada compra llama a una función servidor segura que genera el link de pago.

🔒 SOBRE LA SEGURIDAD DEL ACCESS TOKEN

La Public Key se guarda en Firebase (pública por diseño). El Access Token es secreto y no se guarda en ningún lado de esta app — debe cargarse directamente en Vercel → Environment Variables como MP_ACCESS_TOKEN.
